Aplikasi ini merupakan implementasi Kotlin Coroutines, Flow, dan StateFlow menggunakan arsitektur MVVM dengan Jetpack Compose.
Aplikasi menampilkan daftar berita yang:

1.Update otomatis setiap 2 detik
2.Dapat difilter berdasarkan kategori
3.Menyimpan jumlah berita yang telah dibaca
4.Mengambil detail berita secara asynchronous
